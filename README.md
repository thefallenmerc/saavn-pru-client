This project uses the [Unofficial Saavn APIs](https://github.com/thefallenmerc/saavn-pru-server) and provides a WIP beautiful music app.

### Features

- [ ] Searching songs on saavn
- [ ] Playing songs
- [ ] Seeking song
- [ ] Volume control
- [ ] Add / Remove playlists
- [ ] Play a playlist
- [ ] Next / Previous songs
